nesse exemplo acima vimos a forma mais basica de um formulario com as tags de imput:text que serve para criar a caixa de texto onde será prenchido o fomulario com algum texto

tambem vimos o input:submit que tem uma função "ativa" ele cria um botão para gerar uma açao no formulario, no sosso caso colocamos para enviar.

Tambem vimos o atributo autocomplete da tag form, esse atributo como o nome surgere, server para memorizar o que ja foi digitado dentor das tags de imput:text e surgerir para o proximo usuario quando passar o mouse em cima caso esteja em on, caso queira essa função desativada coloca em off.

Uma tag importanticima que apredemos foi a label que serve para criar uma ligação a palavra que referencia o input, a uma caixa.
